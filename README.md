# Pasta Bar Chabanel

Landing page réalisée pour le restaurant Pasta Bar Chabanel.

## Objectif
Mettre en valeur l’offre de service traiteur du restaurant à travers une landing page claire, attrayante et responsive, orientée conversion.

## Réalisation
- Intégration front end complète
- Structuration du contenu et des sections
- Adaptation responsive (desktop et mobile)
- Attention portée à la lisibilité, à l’ergonomie et à la hiérarchie visuelle

## Technologies utilisées
- HTML
- CSS
- JavaScript
